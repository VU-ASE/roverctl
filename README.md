# Install

## Pre-built binaries

- linux amd64/arm64
- macOS amd64/arm64

Just install roverctl with:

```bash
curl -fsSL https://raw.githubusercontent.com/VU-ASE/roverctl/main/install.sh | bash
```

## From source

1. Clone the repository
2. Run `make build`
3. Execute with `./bin/roverctl`
4. (optional) add the binary to your PATH
